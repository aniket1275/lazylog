# Naming Convention

Rust Variables:
snake_case

Rust Structs:
PascalCase

Rust Constants:
UPPER_SNAKE_CASE

Database Tables:
snake_case plural

API Routes:
RESTful

Example:
GET /api/v1/servers
POST /api/v1/servers

Git Branches:
feature/*
bugfix/*
docs/*
refactor/*

Commits:
feat:
fix:
docs:
test:
refactor:
